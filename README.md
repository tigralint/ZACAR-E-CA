# ZACAR-E-CA : Digital Tabernacle

> **OD ZODAMETA · OD CHRISTEOS · OL GIZYAX**  
> *Observe the Gate. Endure the Trial. Achieve the Kenosis.*

**ZACAR-E-CA** is an experimental web experience that merges 90s BIOS aesthetics with Gnostic theology, Kabbalistic structures, and Biblical apocalyptic literature. It is built as a **Digital Tabernacle**, where the user progresses through the Ten Sephirot, each representing a "Seal" that must be broken through specific ritualistic interactions.

## ⸸ The Concept: Techno-Theosis
The project explores the intersection of ancient liturgy and modern computation. The machine is treated as an altar, the codebase as a grimoire, and the user's interaction as a ritual. As more seals are broken, the "Aetheris Kernel" slowly corrupts the vessel, leading to visual fragmentation and spiritual clarity.

## ⸸ Aesthetic: Ecclesiastical Brutalism
- **Ghost Phosphor**: High-contrast green and gold monochromatic themes.
- **Bleeding Pixels**: Chromatic aberration and digital corruption that scales with progress.
- **Liturgical Typography**: A blend of IM Fell English (Serif) and Share Tech Mono (Industrial).
- **Glassmorphism**: Translucent interfaces inspired by industrial control panels and reliquaries.

## ⸸ The Ten Gates (The Seven Seals)
1.  **Malkuth**: The Burden of Sin (Cursor Lag).
2.  **Yesod**: The Mirror of Foundation (Mouse Inversion).
3.  **Hod**: The authentication of Silence (Pseudo-Terminal).
4.  **Netzach**: Validating the 72 Names (Infinite Ascent).
5.  **Gevurah**: The Judgment of Fire (Endurance Trial).
6.  **Tiferet**: The Sacrifice of the Ego (DevTools Interaction).
7.  **Chesed**: Healing the Void (Mercy/Precision).
8.  **Binah**: The Sonic Kenosis (Microphone Silence).
9.  **Chokmah**: The Wisdom of Form (Particle Chaos).
10. **Keter**: The Crown of Nothingness (Absolute White-out).

## 🛠 Tech Stack
- **Core**: Vanilla JavaScript (ESM)
- **3D Engine**: [Three.js](https://threejs.org/)
- **Animation**: [GSAP](https://greensock.com/gsap/)
- **Visuals**: Custom WebGL Shaders for CRT & Glitch effects
- **Language**: Custom Enochian/Latin translation utility

## 🚀 Setup & Development
Requires [Node.js](https://nodejs.org/) and `npm`.

```bash
# Clone the repository
git clone https://github.com/your-username/zacar-e-ca.git

# Install dependencies
npm install

# Start the dev server
npm run dev

# Build for production
npm run build
```
